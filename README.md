# CT Module 1 Project: Portfolio Landing Page

A responsive personal portfolio landing page built with HTML and CSS as part of the Coding Temple Front End Specialization program.

## Project Overview

This project demonstrates core HTML and CSS skills by building a fully structured portfolio landing page that highlights personal projects and provides a professional web presence.

## Features

- **Navigation Bar** — Logo and anchor links for smooth in-page navigation
- **Hero Section** — Welcome message and introductory content
- **Projects Grid** — Cards showcasing three featured projects with descriptions and GitHub links
- **Footer** — Copyright and site information
- **Responsive Layout** — CSS-based layout designed for clean presentation across screen sizes

## Technologies Used

- HTML5
- CSS3

## Getting Started

To run this project locally, clone the repository and open `index.html` in your browser:

```bash
git clone https://github.com/jenidub/ct-m1-landing-page.git
cd ct-m1-landing-page
open index.html
```
